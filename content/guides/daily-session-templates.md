# Zy AI Academy -- Daily Session Templates

Print these out and keep them at your child's learning station. Check off each session as it is completed.

---

## Ages 5-6 Template

**Expectations:** Shorter sessions. Parent sits with child for all sessions. Focus on building the habit, not perfection. Typing done by parent.

### Daily Checklist

```
Date: _______________    Child's Name: _______________

REQUIRED SESSIONS
[ ] Math Lab (15-20 min)
    Started: _____ Finished: _____
    Topics covered: ________________________________
    How did it go?  Great / OK / Tough

[ ] Language Arts (20-30 min)
    Started: _____ Finished: _____
    Topics covered: ________________________________
    How did it go?  Great / OK / Tough

ELECTIVE SESSIONS (Pick 2)
[ ] Mental Math Wizard (5-10 min)
[ ] Communication Tutor (5 min)
[ ] AI Adventure Tutor (10-15 min)
[ ] Spanish Tutor (10-15 min)
[ ] Creative Thinking (10-15 min)

Elective 1: ____________________
    Started: _____ Finished: _____
    Notes: ________________________________________

Elective 2: ____________________
    Started: _____ Finished: _____
    Notes: ________________________________________

TOTAL TIME TODAY: _____ minutes

Parent Notes:
_______________________________________________
_______________________________________________
```

### Tips for Ages 5-6

- Keep Math Lab under 20 minutes. If they lose focus at 15, stop and celebrate what they did.
- For Language Arts, focus on reading together and simple sentence writing. The tutor will adjust.
- Communication Tutor can be done verbally -- child speaks, parent types.
- Use lots of praise. At this age, showing up is the achievement.

---

## Ages 7-9 Template

**Expectations:** Standard session lengths. Child can begin working more independently. Parent checks in every 10-15 minutes. Child can type simple responses.

### Daily Checklist

```
Date: _______________    Child's Name: _______________

REQUIRED SESSIONS
[ ] Math Lab (20-30 min)
    Started: _____ Finished: _____
    Topics covered: ________________________________
    Problems attempted: _____  Problems correct: _____
    Difficulty level: Easy / Medium / Hard
    How did it go?  Great / OK / Tough

[ ] Language Arts (30-45 min)
    Started: _____ Finished: _____
    Topics covered: ________________________________
    Reading: _____ pages/passages completed
    Writing: _____ sentences/paragraphs written
    How did it go?  Great / OK / Tough

ELECTIVE SESSIONS (Pick 2)
[ ] Mental Math Wizard (10-15 min)
[ ] Communication Tutor (5-10 min)
[ ] AI Adventure Tutor (15-20 min)
[ ] Spanish Tutor (15-20 min)
[ ] Creative Thinking (10-20 min)

Elective 1: ____________________
    Started: _____ Finished: _____
    Score/Completion: _____________________________
    Notes: ________________________________________

Elective 2: ____________________
    Started: _____ Finished: _____
    Score/Completion: _____________________________
    Notes: ________________________________________

TOTAL TIME TODAY: _____ minutes

What I learned today (child fills in):
_______________________________________________

Parent Notes:
_______________________________________________
```

### Tips for Ages 7-9

- Encourage your child to type their own answers. Spelling mistakes are fine -- the AI understands.
- Let them pick electives at the start of the day. Ownership builds motivation.
- If they finish Math Lab fast, ask them to try "challenge mode" by telling the tutor to make it harder.
- Start a small reward system: 5 days in a row earns a special activity on the weekend.

## Ages 10-12 Template

**Expectations:** Full session lengths. Child works independently. Parent reviews at the end of the day. Child manages their own schedule. Challenge modes encouraged.

### Daily Checklist

```
Date: _______________    Child's Name: _______________

REQUIRED SESSIONS
[ ] Math Lab (25-30 min)
    Started: _____ Finished: _____
    Topics: ________________________________________
    Problems attempted: _____  Problems correct: _____
    Difficulty: Standard / Challenge / Advanced
    Self-rating (1-5): _____

[ ] Language Arts (30-60 min)
    Started: _____ Finished: _____
    Topics: ________________________________________
    Reading comprehension score: _____
    Writing piece: _________________________________
    Self-rating (1-5): _____

ELECTIVE SESSIONS (Pick 2)
[ ] Mental Math Wizard (10-15 min) -- Target speed: _____ sec/problem
[ ] Communication Tutor (10 min) -- Topic practiced: _______________
[ ] AI Adventure Tutor (15-20 min) -- Decisions made: _____
[ ] Spanish Tutor (15-20 min) -- New words learned: _____
[ ] Creative Thinking (15-20 min) -- Idea generated: _______________

Elective 1: ____________________
    Time: _____ min    Score/Notes: ________________

Elective 2: ____________________
    Time: _____ min    Score/Notes: ________________

TOTAL TIME TODAY: _____ minutes

Today's Reflection (child fills in):
What went well: ___________________________________
What was hard: ____________________________________
Tomorrow I want to: _______________________________

Parent Review (evening):
_______________________________________________
```

### Tips for Ages 10-12

- Let them set their own schedule within the required/elective framework.
- Encourage them to ask the tutor to explain concepts they do not understand instead of skipping.
- Challenge mode in Math Lab should be the default by week 3.
- Have them teach you something they learned each day. Teaching reinforces learning.
